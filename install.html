Előfeltételek

⚠️ Fontos: A MITE:BE nem kompatibilis a modern Minecraft verziókkal. Pontosan Minecraft 1.6.4-et igényel, Forge nélkül.

1 Java 8 telepítése

A Minecraft 1.6.4 Java 8-at igényel — újabb verzió (Java 11, 17, 21 stb.) nem működik!

  1. Töltsd le a Java 8 JRE-t innen: adoptium.net — Eclipse Temurin 8
  2. Válaszd az operációs rendszerednek megfelelő verziót (Windows x64 → .msi ajánlott)
  3. Telepítsd a szokásos módon — jegyezd meg a telepítési helyet!
💡 Ha már van Java-d telepítve, a java -version parancs a terminálban megmutatja a verziót. "1.8.0"-val kell kezdődnie.

2 PrismLauncher telepítése

  1. Töltsd le a PrismLauncher-t innen: prismlauncher.org
  2. Telepítsd és indítsd el
  3. Adj hozzá egy Minecraft fiókot (Microsoft fiók szükséges)
  4. Beállítások → Java → kattints az "Auto-detect" gombra és válaszd ki a Java 8 (1.8.0) telepítést
⚠️ Ha nem állítod be a Java 8-at a PrismLauncherben, a mod nem fog működni — a leggyakoribb hiba ez!

MITE:BE Letöltése

A mod egy teljes PrismLauncher instance formájában érhető el — minden benne van (Minecraft 1.6.4 + mod JAR-ok + textúracsomag).

⬇ MITE:BE v0.9.0 hf3 letöltése ZIP fájl · ~336 MB · GitHub Release
⚠️ Ne csomagold ki a ZIP fájlt! A PrismLauncher közvetlenül importálja — az egész ZIP kell egyben!

Telepítés (Importálás)

  1. Nyisd meg a PrismLauncher-t
  2. Kattints a "Új Instance" (Add Instance) gombra
  3. Válaszd az "Importálás" (Import) fület
  4. Tallózd ki a letöltött MiTE-BE_v0.9.0_hf3.zip fájlt
  5. Kattints az OK gombra — a PrismLauncher automatikusan mindent beállít
  6. Kattints duplán az új instance-ra az indításhoz

Ha a Minecraft főmenü megjelenik a MITE:BE logóval — kész, sikeres a telepítés!

Haladó: Kézi telepítés (instance nélkül)

Ha a ZIP importálás nem működik, vagy kézzel szeretnéd beállítani:

  1. PrismLauncher: Új Instance → Minecraft verzió: 1.6.4 (Vanilla, Forge nélkül!)
  2. Csomagold ki a ZIP-et egy ideiglenes mappába
  3. Jobb klikk az instance-on → Mods szerkesztése → add hozzá a .jar fájlokat a .minecraft/ mappából
  4. Másold át a resourcepacks és config mappákat is az instance .minecraft mappájába
  5. Indítsd el az instance-ot

Első Indítás Tippek

  • Játékmód választás: A MITE:BE két módot kínál — Simple (fáklyák nem alszanak ki, könnyebb) és Hardcore (fáklya kialszik, nehezebb). Kezdőknek a Simple ajánlott!
  • Biome választás: A Magasföld (Highland) biome az ajánlott kezdő biome — lapos terep, jó erőforrások, nincs extrém veszély.
  • Wiki: Tartsd nyitva a Fázis Útmutatót — lépésről lépésre végigvezet a progresszión!
💡 A Minecraft 1.6.4 régi verzió — ne lepődj meg a régebbi grafikán és menürendszeren!

Prerequisites

⚠️ Important: MITE:BE is not compatible with modern Minecraft versions. It requires exactly Minecraft 1.6.4, without Forge.

1 Install Java 8

Minecraft 1.6.4 requires Java 8 — newer versions (Java 11, 17, 21, etc.) will not work!

  1. Download Java 8 JRE from: adoptium.net — Eclipse Temurin 8
  2. Select the version for your OS (Windows x64 → .msi recommended)
  3. Install normally — remember the installation path!
💡 If you already have Java installed, run java -version in a terminal. It should start with "1.8.0".

2 Install PrismLauncher

  1. Download PrismLauncher from: prismlauncher.org
  2. Install and launch it
  3. Add a Minecraft account (Microsoft account required)
  4. Settings → Java → click "Auto-detect" and select the Java 8 (1.8.0) installation
⚠️ If you don't set Java 8 in PrismLauncher, the mod will not work — this is the most common issue!

Download MITE:BE

The mod is distributed as a complete PrismLauncher instance — everything is included (Minecraft 1.6.4 + mod JARs + texture pack).

⬇ Download MITE:BE v0.9.0 hf3 ZIP file · ~336 MB · GitHub Release
⚠️ Do not extract the ZIP file! PrismLauncher imports it directly — the whole ZIP is needed as-is!

Installation (Import)

  1. Open PrismLauncher
  2. Click the "Add Instance" button
  3. Select the "Import" tab
  4. Browse to the downloaded MiTE-BE_v0.9.0_hf3.zip file
  5. Click OK — PrismLauncher will automatically set everything up
  6. Double-click the new instance to launch

If the Minecraft main menu appears with the MITE:BE logo — you're done, installation successful!

Advanced: Manual Installation (without instance import)

If the ZIP import doesn't work, or you prefer manual setup:

  1. PrismLauncher: Add Instance → Minecraft version: 1.6.4 (Vanilla, no Forge!)
  2. Extract the ZIP to a temporary folder
  3. Right-click the instance → Edit Mods → add the .jar files from the .minecraft/ folder
  4. Copy the resourcepacks and config folders into the instance's .minecraft folder
  5. Launch the instance

First Launch Tips

  • Game mode: MITE:BE offers two modes — Simple (torches don't expire, easier) and Hardcore (torches expire, harder). Simple is recommended for beginners!
  • Biome choice: The Highland biome is the recommended starting biome — flat terrain, good resources, no extreme dangers.
  • Wiki: Keep the Phase Guide open — it walks you through the progression step by step!
💡 Minecraft 1.6.4 is an old version — don't be surprised by older graphics and menus!

Troubleshooting

ProblemSolution
"Java version error" / crash on startup PrismLauncher → Settings → Java → verify that Java 8 (1.8.0) is selected. NOT Java 17 or 21!
Black screen / doesn't load 1) Check Java 8 setting. 2) Settings → Java → Min memory: 512 MB, Max memory: 2048 MB. 3) Try restarting.
"Mod won't load" MITE:BE does NOT use Forge! Install on Vanilla 1.6.4. If Forge is in the instance, remove it.
ZIP import doesn't work Make sure you selected the original .zip file (not the extracted folder). If it still fails, try manual installation (above).
Missing textures Settings → Resource Packs → verify the MITE:BE texture pack is active.
