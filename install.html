Telepítési Útmutató

A MITE:BE egy Minecraft 1.6.4-es mod. Ez az útmutató PrismLauncher-rel való telepítést mutat be — ez az ajánlott módszer.

⚠️ Fontos: A MITE:BE nem kompatibilis a modern Minecraft verziókkal. Pontosan Minecraft 1.6.4-et igényel, Forge nélkül.

1. Java 8 telepítése

A Minecraft 1.6.4 Java 8-at igényel — Java 11 vagy újabb nem működik megfelelően.

  1. Töltsd le a Java 8 JRE-t (Adoptium / Eclipse Temurin ajánlott: adoptium.net)
  2. Telepítsd szokásos módon

2. PrismLauncher telepítése

  1. Töltsd le a PrismLauncher-t: prismlauncher.org
  2. Indítsd el és adj hozzá egy Minecraft fiókot
  3. Beállítások → Java → válaszd ki a Java 8-as telepítést

3. MITE:BE Instance létrehozása

  1. PrismLauncher: Új Instance → Minecraft verzió: 1.6.4 (Vanilla, Forge nélkül)
  2. Töltsd le a MITE:BE modot: r/MITE subreddit vagy a MiTE Discord
  3. Jobb klikk az instance-on → Mods szerkesztése → add hozzá a .jar fájlt
  4. Indítsd el az instance-ot — a főmenü megjelenik ha minden rendben van
💡 Közösség: A MiTE Discord-on (discord.gg/7myyGAXJ6v) megtalálod a legfrissebb verziót, textúracsomagokat és kiegészítőket. Az r/MITE subreddit szintén aktív forrás.

Installation Guide

MITE:BE is a Minecraft 1.6.4 mod. This guide covers installation using PrismLauncher — the recommended method.

⚠️ Important: MITE:BE is not compatible with modern Minecraft versions. It requires exactly Minecraft 1.6.4, without Forge.

1. Install Java 8

Minecraft 1.6.4 requires Java 8 — Java 11 or newer does not work correctly.

  1. Download Java 8 JRE (Adoptium / Eclipse Temurin recommended: adoptium.net)
  2. Install it normally

2. Install PrismLauncher

  1. Download PrismLauncher at prismlauncher.org
  2. Launch it and add your Minecraft account
  3. Settings → Java → select your Java 8 installation

3. Create the MITE:BE Instance

  1. In PrismLauncher: Add Instance → Minecraft version: 1.6.4 (Vanilla, no Forge)
  2. Download the MITE:BE mod from the community: r/MITE subreddit or the MiTE Discord
  3. Right-click the instance → Edit Mods → add the .jar file
  4. Launch the instance — the main menu appears if everything is correct
💡 Community: Join the MiTE Discord (discord.gg/7myyGAXJ6v) to find the latest version, texture packs, and add-ons. The r/MITE subreddit is also an active resource.
