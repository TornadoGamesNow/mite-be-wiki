# MITE: Break Everything — Wiki & Progression Guide

The first public wiki for **MITE: Break Everything** — a hardcore total-conversion Minecraft mod (v0.9.0 hf3).

🌐 **Live site:** https://tornadogamesnow.github.io/mite-be-wiki

---

## What's inside

- Bilingual: Magyar 🇭🇺 / English 🇬🇧
- Full progression guide — Early game → Mid game → Endgame checklists
- Biome guide — where to start, what to watch out for
- Phase milestones — what unlocks what
- Weapon, armor & enchanting reference tables
- Ore processing, furnaces, alloys
- Workbench tiers and crafting recipes
- Professions and skills
- XP requirements, item quality tiers
- Mob XP drops, food values, stack limits, and more

## Tech

Static HTML/CSS/JS — no server, no build tools, no dependencies.
